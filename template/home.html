API de Previsão de Ações

Esta API foi desenvolvida como solução para o Tech Challenge - Fase 4.

Utilizamos um modelo de Redes Neurais Long Short-Term Memory (LSTM) para prever valores de fechamento de ações, e o FastAPI para servir este modelo como uma API RESTful.
